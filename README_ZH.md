没关系！本模组来帮你！

## ✨ 核心功能

加载本模组后，<u>**除非胜利 / 放弃**</u>，否则存档**不会被删除**。

从此告别胡局暴毙！

> 多人模式下，必须**房主加载**才会生效。同理，房主外队友可不必加载。

## 🎮 玩家安装

1. 从 **Releases** 页面下载最新的 **zip** 压缩包。
2. 解压并将内部的 `DeathNoDelete` 文件夹整体复制到游戏的 `<Slay the Spire 2>/mods/` 目录下。
3. 启动游戏并在 **Mods** 菜单中勾选启用本模组。

## ⚙️ 编译

如果你需要自己编译：
- 将 `DeathNoDelete.csproj` 的 `Sts2Dir` 属性设置为你的游戏路径。
- 或编译时使用: `dotnet build -p:Sts2Dir="你的游戏路径"`
